# HDMISwitchControl

Allows for an ESP32 to receive HTTP requests and send IR commands to the [Monoprice Blackbird 8K Dual Function Splitter/Switch](https://www.monoprice.com/product?p_id=44086) 

IR Codes

hdmi_one = 0x40BF10EF;

hdmi_two = 0x40BF50AF;

power = 0x40BF00FF;

toggle = 0x40BF708F;
